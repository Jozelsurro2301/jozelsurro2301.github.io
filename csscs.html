SELECTORS

Used to select HTML elements you want to style

Selectors Description Syntax
Universal Selctor This selects all the elemets in your HTML File.
Type This selects HTML elemets in you HTML File.
ID This selects elements based on its unique identifier/ID attribute.
Class This selects elements based on its class Syntax.
Combinators A complex selector with more than one element that has a relationship to each other.
Attribute Selects HTML elements that has a specific attribute.
Pseudo Defines a special state of an element and adds a style to it based on its state.

FONT PROPERTIES

Sets the fonts contents style based on the given attribute.

Property Description Syntax
font-family Specifies what font family will be used.
font-style Used to style the text into normal, bold, italic, and many more.
font-weight Used to specify the thickness of the font.
font-size Used to specify the size of the text.
color Used to specify the color of the text

TEXT PROPERTIES

Property Description Syntax
text-alignment Used to set the horizontal alignment of the text as left, right, centered or justified.
text-decoration Used to add or remove text decorations like underline, overline, line-trough, or none.
text-transform Used to change the case of the text.
line-height Used to specify the space between the lines of the text block.

BACKGROUND PROPERTIES

Property Description Syntax
background-color Used to specify the background color of the page.
background-image Used to specify one or more background images to a page
background-position Used to specify the positioning of the image on the page.
background-attachment Used to specify the attachment of the image with respect to the container.

BOX PROPERTIES

Property Description Syntax
border Sets the elements border width, color and size.
height Sets the elements height as a length.
width Sets the elements width as a length.
padding Sets the elements space between the border and the container.
margin Sets the elements margin by specifying its length

CLASSIFCATION PROPERTIES

Specifies how and where an element is displayed on a page.

Property Description Syntax
display Defines how elements are displayed in the web page.
position Specifies the positioning method of an element.
float Defines the flow of contect by determining if an element floats left or right, allowing text, or image to wrap around it or be displayed in-line.

POSITION ATTRIBUTES

Attribute Description
fixed Property will be position relative to the viewport. The element will remain at the same position even you scroll the page.
static The default positioning method. The element will be position with the normal flow of the page.
relative Positioned relatively with the other elements which are sittinh at the top of it.
absolute Positioned with respect to its nearest relative or non-static parent or ancestor.

DISPLAY ATTRIBUTES

Attribute Description
inline Displays an element as an inline element.
block Displays an element as a block element.
flex Displays an element as a block-level flex container.
grid Displays an element as a block-level grid contianer.
inline-block, inline-flex, inline-grid Displays an element as an inline-level container.
none Used to remove the element.
initial Used to set the default value.
inherit Used to inherit properties from its parents' elements

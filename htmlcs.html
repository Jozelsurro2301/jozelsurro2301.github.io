HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

<link>

Often used to link to external style sheet.

<meta>

Defines the metadata(information) of the HTML document.

<title>

Defines the title of the HTML.

<h1> - <h6>

Heading elements that help to organize your website. It represent the six levels of section headings with h1 being the highest section level and h6 as the lowest.

<div>

Defines a division in an HTML document. It is also used as a container for HTML elements

<span>

Used as an inline container to mark up a part of a text.

<p>

Defines a paragraph.

<br>

Inserts a line break. This tag is an empty tag which means it has no end tag

<hr>

Displays a horizontal line that is used to separate content in an HTML page.

<ol>

Defines an ordered list. It can be numerical or alphabetical.

<ul>

Defines an unordered list.

<li>

Defines the list item. Used inside an ordered list and unordered list.

<form>

Creates a HTML form for user input. Common input fields are action and method parameters.

<fieldset>

Used to group related elements in a form.

<legend>

Used to add a caption to a group of related form elements that have been grouped by the fieldset element.

<select>

Used to create a drop-down list.

<option>

Used inside the select element to list down the options for the drop-down list.

<input>

Used to create interactive controls for web-based forms in order to accept data from the user. A wide variety of types of input are available like text, button, checkbox, password, email, radio, submit, and many more!

<textarea>

Defines a multi-line text input control. Often used to get user inputs like comments and feedback.

<button>

Defines a clickable button.
